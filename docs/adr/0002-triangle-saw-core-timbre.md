# Triangle/saw blend as the core timbre

The oscillator swarm will use a triangle/saw blend as its base timbre. This keeps the sound warm and organic while retaining enough harmonic density to feel thick and lush.

**Considered options**: saw only, triangle/saw blend, sine/triangle with soft saturation, constrained morphable shape, multiple selectable waveforms.

**Decision**: triangle/saw blend.

**Consequence**: the design should not become a waveform menu or generic oscillator. Timbre controls, if any, should shape the triangle/saw balance or brightness around this core identity.
