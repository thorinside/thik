# Thickness as a single swarm macro

The oscillator will expose a single Thickness control for making the swarm denser and more animated. Thickness should increase the apparent voice prominence, detune width, and organic movement together rather than exposing separate voice-count, detune, and drift controls.

**Considered options**: fixed medium swarm, Thickness macro, Density control, separate Voice Count/Detune/Drift/Spread controls, preset modes.

**Decision**: single Thickness macro.

**Consequence**: the plugin stays focused as one curated oscillator sound instead of becoming a general oscillator-bank editor.
