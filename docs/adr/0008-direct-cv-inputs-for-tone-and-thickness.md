# Direct CV inputs for Tone and Thickness

The oscillator will expose direct CV input parameters for both `Tone` and `Thickness`. Although the disting NT can map CV to parameters, direct CV inputs make the two main sound macros immediately patchable and visible in the plugin's control surface.

**Considered options**: panel-only Tone/Thickness, CV for Thickness only, CV for both Tone and Thickness, defer CV inputs.

**Decision**: add direct CV inputs for both Tone and Thickness.

**Consequence**: the parameter set includes routing parameters for `Tone CV` and `Thickness CV`, increasing the surface slightly in exchange for better modular playability.
