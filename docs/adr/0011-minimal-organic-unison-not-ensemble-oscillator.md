# Minimal organic unison, not an ensemble oscillator

This plugin is distinct from `nt_enosc` by being a minimal organic unison oscillator: one pitched swarm with no scale, chord, interval, freeze, warp, twist, or ensemble-engine modes. It is also warmer and more triangle/saw focused, with playability centered on the `Tone` and `Thickness` macros.

**Considered options**: emphasize minimal unison, emphasize warmer triangle/saw swarm, emphasize macro playability, or all of these.

**Decision**: all of these, with minimal organic unison as the primary distinction.

**Consequence**: implementation and future feature requests should reject scale/chord/mode expansion unless the project is deliberately re-scoped.
